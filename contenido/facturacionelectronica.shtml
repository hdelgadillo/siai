Facturación Electrónica

Para contar con la facturación electrónica (CFDI) se necesita:

1: Firma Electrónica Avanzada (Fiel)

La Fiel es un comprobante digital de identidad que se utiliza para el registro de los contribuyentes. Se obtiene con cita en las oficinas del SAT donde se registran huellas dactilares, retina y otros elementos que evitan el robo de identidad. En caso de no tener la Fiel se puede obtener vía el SAT en el siguiente enlace. Si ya se cuenta con una y no se tiene a la mano los archivos que se generan durante la cita (usualmente se entregan en una memoria USB personal) en este link se pueden volver a descargar. Esto es necesario para proceder con la facturación electrónica.

2: Certificado de Sello Digital

Este sello es una medida adicional que avala la impresión del comprobante electrónico. Para generarlo se requiere de la Fiel y es un proceso breve que se hace a través de una herramienta de software desarrollada por el SAT. El programa Solcedi es un software que genera el sello digital. Se puede instalar en Windows, Mac OS X y Linux; una vez abierto, se debe elegir la opción “Solicitud de Certificados de Sello Digital” y seguir los pasos para subir el archivo de la Fiel y concluir con este procedimiento. Los archivos generados servirán para la facturación electrónica. Si se tienen dudas, en este enlace se muestra los pasos para obtener el sello. El detalle del CSD está a se encuentra a partir de la página 19.

3: Factura Electrónica

Hecho esto, ya se puede pasar a la creación de CFDI. El Servicio de Administración Tributaria ofrece una herramienta para los contribuyentes que funciona en línea; o puede contratar los servicios de un PAC autorizado por el SAT, o si lo desea podemos nosotros podemos ayudarte durante todo el proceso interactuando con cada una de las instancias necesarias para obtener la facturación.
